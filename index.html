Was ist Django Girls?

Django Girls ist ein kostenfreier Workshop der Frauen jeden Alters eine Einführung ins Programmieren bietet.

Während zwei Tagen wird eine eigene Website konstruiert, auf der ein Blog publiziert werden kann. Dazu werden verschiedene Tools, Frameworks und Programmiersprachen verwendet. Im Folgenden sind einige der erstellten Websites aufgeführt:
